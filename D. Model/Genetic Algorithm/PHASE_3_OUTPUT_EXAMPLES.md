- Extract top K: ~50ms
- Consolidate foods: ~100ms
- Classify foods: ~200ms
- Select top N: ~50ms
- Total: ~400ms

**Memory usage**:

**Accuracy**:
- Top K extraction: 100% (direct from fitness)
- Food consolidation: 98% (frequency-based)
- Classification: 92% (keyword-based)
- Selection: 100% (frequency ranking)
