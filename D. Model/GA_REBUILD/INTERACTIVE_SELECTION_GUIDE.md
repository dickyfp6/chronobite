```python
selected_meal = []

for slot_idx, slot_name in enumerate(SLOT_NAMES):
    options = slot_options.get(slot_name, [])
    
    # Display options
    for i, option in enumerate(options, 1):
        food_name = option.get('food_name', 'Unknown')
        energy = option.get('energy_kcal', 0)
        protein = option.get('protein_g', 0)
        print(f"{i}. {food_name:30} | Energy: {energy:6.1f} kcal | Protein: {protein:5.1f}g")
    
    # Get user choice
    choice = int(input(f"Pilih opsi (1-{len(options)}) [default=1]: ")) - 1
    selected_meal.append(options[choice])
```

**STEP 8: Nutrition Analysis**
```python
selected_df = pd.DataFrame(selected_meal).reset_index(drop=True)
selected_nutrition = calculate_total_nutrition(selected_df)

# Compare against guidelines
for nutrient_col, unit, label in key_nutrients:
    value = selected_nutrition[nutrient_col]
    constraint = guidelines.get(nutrient_col, {})
    min_val = constraint.get('min', 0)
    max_val = constraint.get('max', float('inf'))
    
    # Check compliance
    if value < min_val:
        status = "🔴 LOW"
    elif value > max_val:
        status = "🟡 HIGH"
    else:
        status = "🟢 OK"
```

## 🎯 Input Validation

### User Input Handling:

```python
while True:
    try:
        choice_str = input(f"Pilih opsi (1-{len(options)}) [default=1]: ").strip()
        
        # Default to 1 jika Enter saja
        if choice_str == "":
            choice = 0
        else:
            choice = int(choice_str) - 1
        
        # Validate range
        if 0 <= choice < len(options):
            selected_item = options[choice].copy()
            selected_meal.append(selected_item)
            break
        else:
            print(f"✗ Pilih antara 1-{len(options)}")
    
    except ValueError:
        print("✗ Input harus berupa angka")
```

**Features:**
- ✅ Empty input defaults to 1
- ✅ Non-numeric input shows error
- ✅ Out-of-range shows error
- ✅ Repeats until valid input

## 📊 Compliance Status Indicators

| Scenario | Status | Symbol | Meaning |
|----------|--------|--------|---------|
| Within min-max | OK | 🟢 | Good |
| Below minimum | LOW | 🔴 | Need more |
| Above maximum | HIGH | 🟡 | Too much |

### Compliance Rate Calculation:

```
Compliance Rate = (Count of 🟢 OK) / (Total nutrients) × 100%

If Compliance >= 100%: ✅ ALL nutrients within guidelines!
If Compliance >= 75%:  ⚠️ Most nutrients are within guidelines
If Compliance < 75%:   ❌ Several nutrients outside guidelines
```
